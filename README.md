  ## :information_source: How To Use

  ```bash
  # Clone this repository
  $ git clone https://github.com/eltonlazzarin/facebook-comments-layout.git

  # Go into the repository
  $ cd facebook-comments-layout

  # Install dependencies
  $ yarn

  # Run the app
  $ yarn start
  ```
